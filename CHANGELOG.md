# Changelog

All notable public changes will be documented here.

## [0.1.0-alpha.1] - Unreleased (Developer Preview)

### Added

- Local encrypted vault and local unlock lifecycle.
- Provider-neutral AI capability interfaces with MockProvider and OpenAIProvider implementations.
- Chronological transcript, translation, segment insight, meeting analysis, communication feedback, minutes, and recording comparison artifacts.
- Real-time microphone and macOS system-audio capture with encrypted recovery spool.
- Uploaded audio/video/transcript/subtitle processing and FFmpeg integration boundary.
- Immutable artifact versions, controlled regeneration, local library, history, and selected-version export.
- Public documentation, contribution guidance, security policy, and self-contained public contract tests.
- Reproducible public-source validation and dependency-license review.

### Known limitations

See [docs/KNOWN_LIMITATIONS.md](docs/KNOWN_LIMITATIONS.md). Live OpenAI API smoke testing, signed/notarized macOS distribution, and full Windows system-audio support remain pending.
