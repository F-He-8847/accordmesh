# Asset Provenance

## AccordMesh application icon

The application icon under `apps/desktop/src-tauri/icons/` is custom project artwork created specifically for AccordMesh through a generative design workflow and selected by the AccordMesh maintainer. It is not intentionally derived from a third-party stock icon, font file, or trademarked logo. The maintainer has approved its inclusion, modification, and redistribution with this project.

The committed PNG sources have been re-encoded to remove embedded editing metadata. Maintainers should retain the selected source artwork and regeneration process outside the repository so future icon revisions remain traceable.

## Screenshots and sample data

Future screenshots must use fictional projects and redact operating-system usernames, local paths, credentials, meeting content, and notifications. Test transcript and subtitle files must be clearly fictional and must not be derived from private meetings.
