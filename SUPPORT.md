# Support

AccordMesh is currently a Developer Preview maintained on a best-effort basis.

Use GitHub Issues for reproducible bugs and scoped feature requests. Before filing an issue, search existing issues and review [Known Limitations](docs/KNOWN_LIMITATIONS.md). General support, setup help, and provider-compatibility questions are handled on a best-effort basis and may be redirected to existing documentation.

Never post provider credentials, real meeting content, vault files, databases, media, or unredacted logs. Security issues must follow [SECURITY.md](SECURITY.md).

The project does not currently provide guaranteed response times, paid support, a hosted service, or recovery for lost local passwords.
