# AccordMesh Evaluations

This directory is reserved for reproducible, fictional evaluation datasets and scoring tools.

Evaluation work should measure at least:

- source transcript preservation;
- translation fidelity for names, numbers, dates, negation, conditions, and uncertainty;
- evidence-reference validity;
- separation of facts, inference, ambiguity, and advice;
- avoidance of invented speakers, owners, deadlines, decisions, and commitments;
- avoidance of unsupported psychological, deception, emotion, or hidden-motive claims;
- consistency across retries and provider/model changes.

Do not add real meeting material. Every fixture must document its fictional origin, language, expected properties, and license.
